Bienvenue chez Logos ! Nous sommes ravis que vous nous rejoigniez pour façonner le web décentralisé. Que vous soyez développeur, rédacteur, designer, ou que vous apportiez d'autres compétences précieuses, il existe de nombreuses façons de vous impliquer et d'avoir un impact.

Pour commencer, vous pouvez soumettre vos idées ou projets via le [formulaire de proposition](/proposals).

Vous pouvez également explorer des guides personnalisés en fonction de vos compétences :

- [Pour contribuer à des initiatives locales de terrain](/resources/activists)
- [Pour les Développeurs](/resources/developers)
- [Pour les Rédacteurs](/resources/writers)
- [Pour les Organisateurs d'Événements](/resources/event-organisers)
- [Pour les Traducteurs](/resources/translators)
- [Pour les Designers](/resources/designers)
